🛰️ NDVI Analysis — Mallorca

Seasonal Vegetation Health Analysis using Landsat 8 Satellite Imagery

📊 Project Overview

This geospatial analysis project compares vegetation health (NDVI) between summer and winter on Mallorca using Landsat 8 Collection 2 Level-2 satellite data. The study captures how 2026's above-average Mediterranean precipitation translates into vegetation patterns, providing insights into seasonal vegetation dynamics and climate resilience.

🌍 Geospatial Analysis

Multi-spectral satellite imagery processing with automated mosaicking, clipping, and reprojection to administrative boundaries.

📈 NDVI Calculation

Normalized Difference Vegetation Index computation from Landsat 8 bands, generating continuous and classified rasters.

🗺️ Seasonal Comparison

Summer vs. Winter analysis revealing vegetation response to Mediterranean rainfall patterns in a wet year.

💾 Multi-Format Export

Outputs in GeoTIFF (raster), GeoPackage (vector), and PNG (visualization) for diverse GIS workflows.

NDVI values are classified into five distinct vegetation categories:

Class NDVI Range Color Meaning

🌧️ 2026: A Wet Year

Why This Analysis Matters

  • Above-average winter precipitation across the Mediterranean led to exceptional vegetation productivity in 2026
  • Seasonal greening response: Areas that respond strongly to rainfall show pronounced summer NDVI peaks when moisture from winter rains fuels photosynthetic activity
  • Climate resilience baseline: This wet-year analysis provides a reference point for comparing future droughts or anomalous precipitation events
  • Land management insights: Identifying which ecosystems are most water-responsive helps prioritize conservation and irrigation planning
  • Urbanization patterns: Built-up areas show minimal seasonal variation, highlighting the ecological impact of development

🛠️ Technology & Workflow

Python 3.9+ Rasterio (raster I/O) Geopandas (vector data) NumPy (numerical computing) Matplotlib (visualization) Shapely (geometry ops) Landsat 8 Collection 2 L2

Workflow Steps:

  1. Auto-detect and categorize satellite scenes by season (May–Oct summer, Nov–Apr winter)
  2. Load Landsat 8 Red (B4) and NIR (B5) bands for multiple scenes
  3. Mosaic multiple tiles into seamless coverage
  4. Clip to island administrative boundaries
  5. Apply Landsat C2 L2 scale factors (uint16 → surface reflectance)
  6. Compute NDVI: (NIR − Red) / (NIR + Red)
  7. Calculate seasonal difference: Summer NDVI − Winter NDVI
  8. Classify into five vegetation categories
  9. Export continuous rasters (GeoTIFF) and classified vectors (GeoPackage)
  10. Generate publication-ready visualizations (PNG)
